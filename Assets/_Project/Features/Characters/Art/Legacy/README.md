# Legacy Character Art

This folder is the long-term home for imported packs, archive sprites, and older character art that is not part of the current canonical production pipeline.

Current rule:
- treat the existing `Animations`, `Licenses`, `SourceFiles`, and `Sprites` folders as legacy/archive content unless a specific asset is actively promoted into production use
- do not move large legacy art folders blindly; migrate only when references are understood

Canonical production-facing art should go under sibling `Production` and `Source` folders instead.

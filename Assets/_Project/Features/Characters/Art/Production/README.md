# Production Character Art

This folder is the canonical home for character art that the live game is expected to use.

Intended subfolders:
- `PlayableSprites`
- `Animations`
- `Portraits`
- `EventVignettes`

Rules:
- only approved, in-use, or near-ship assets belong here
- do not use this as a dumping ground for trials, prompts, or imported packs
- anything generated by Gemini, PixelLab, or edited in Aseprite should land here only after review/approval

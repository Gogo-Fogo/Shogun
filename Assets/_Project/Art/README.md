# _Project Art

This folder is in a transition state.

Use these rules:

- Existing `Backgrounds/` and `References/` content should be treated as legacy/archive by default unless explicitly promoted.
- New editable AI-assisted or manual working files should go under `Source/`.
- New approved runtime-facing art should go under `Production/`.
- `Legacy/` is the long-term home for imported packs and older exploration content when it is worth separating later.

Do not mass-move the current art tree blindly. Migrate only when references and usage are understood.
